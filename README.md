# Secure Remote Access Solution

## Project Overview

Designed and implemented an enterprise-grade secure remote access solution simulating a real-world remote workforce environment. The solution integrates VPN, Multi-Factor Authentication (MFA), Identity and Access Management (IAM), and Zero Trust principles to ensure secure access to organizational resources.

## Objectives

* Establish encrypted remote access using IPsec/IKEv2 VPN through strongSwan on Ubuntu Server
* Enforce Multi-Factor Authentication (MFA) for secure user verification
* Apply Identity and Access Management (IAM) and Role-Based Access Control (RBAC)
* Implement Zero Trust security principles
* Secure remote desktop access for employees

## Tools & Technologies Used

* Ubuntu Server
* Oracle VirtualBox
* strongSwan (IPsec/IKEv2 VPN)
* RustDesk
* Multi-Factor Authentication (MFA)
* Identity and Access Management (IAM)
* Role-Based Access Control (RBAC)
* Wireshark
* Linux Administration

## What Was Done

* Configured a secure IPsec/IKEv2 VPN using strongSwan on Ubuntu Server
* Implemented Multi-Factor Authentication to strengthen user authentication
* Applied IAM and RBAC policies to restrict access based on user roles
* Configured RustDesk for secure remote desktop connectivity
* Validated encrypted communication and remote connectivity
* Monitored and verified network traffic using Wireshark
* Documented security controls, access policies, and implementation results

## Outcome

Successfully built and validated a secure remote access environment within a virtual lab. All remote access sessions were authenticated, encrypted, and controlled through role-based access restrictions in accordance with Zero Trust security principles.
