# Secure Remote Access Solution

## Project Overview
Designed and implemented a secure remote access solution for employees using VPN, MFA, IAM, and Zero Trust principles.

## Objectives
- Secure remote employee access
- Protect company resources
- Implement secure authentication

## Tools Used
-Ubuntu Server
-IPsec/IKEv2 VPN
-RustDesk
-Multi-Factor Authentication (MFA)
-Identity and Access Management (IAM)
-Linux Administration

## Activities Performed
- Configured secure remote access
- Tested connectivity
- Implemented security controls
- Evaluated security risks

## Outcome
Successfully designed and tested a secure remote access solution in a virtual environment.
